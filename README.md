# Chess Game

**Tech stack:** Java

This is a text-based chess game implemented in Java that follows standard chess rules. The game runs in the terminal and supports advanced rules such as castling, en passant, pawn promotion, and the 50-move rule.

## Features
- Full chess rule enforcement
- Check, checkmate, and stalemate detection
- Move tracking and game state evaluation
- Save game history to file

## Design Notes
- Object-oriented design with clear separation of responsibilities
- Rule validation handled at the move level
- Game state evaluation separated from user input handling
